# Roadmap Sync

Use this skill when the task is to re-plan the next phase of the study or convert current evidence into a prioritized research roadmap.

## Workflow

1. Read `AGENTS.md`.
2. Reconcile the benchmark truth first.
3. Read:
   - `docs/PUBLICATION_STRATEGY_2026-03-22.md`
   - `docs/BENCHMARK_MATRIX_2026-03-22.md`
   - `docs/BENCHMARK_SUMMARY.md`
   - `paper/draft.md`
4. Build a roadmap with:
   - near-term publishable track
   - broader research track
   - exact next task order
   - platform and compute choice
   - stop conditions
5. If the roadmap spans many workflows, spawn `benchmark_strategist`.

## Repository Caveats

- Do not let the roadmap drift back into a generic quantum-win narrative.
- Publication-safe planning should assume the strongest current evidence favors classical baselines.
